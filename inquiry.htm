자동조판시스템
서비스 지원 범위
디자인 기본 제공 - 기사관리, 지면관리, 기사입력, 광고관리, 템플렛, 편집/교열, 모니터링
맞춤 변경 - 사전 기획 협의
- 스크립트 로직 변경이 병행되는 경우 작업 일정 및 범위를 확인하여 추가 작업
API 기본 API - 컨설팅 후 바로 제공
맞춤 API - 이외 필요한 API 요청 시 작업 가능 여부를 확인한 후 제작하여 제공
지원 개발 - 각종 가이드 문서 전달
- 스크립트 오류 수정, 충돌된 영역 수정
- 방문 지원(단, 내부지원이 불가능한 사안에 한함)
- 설치 후 모니터링
- 업무 외 시간 긴급상황 발생 시 조치
운영 - 최신 사례, 유사 사례 전달
- 서비스 개선 사항 전달
- 업무 외 시간 긴급사항 발생 시 커뮤니케이션
※ 서비스 개시 후 6개월간 미디어토마토 댓글모음/토마토AD 서비스가 전면 무료로 제공됩니다.
이후 일평균 PV 1000 초과 시 광고가 삽입 될 수 있으며, 삽입된 광고는 소액결제를 통해 삭제 가능합니다.
